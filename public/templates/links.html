Links

W3Schools http://www.w3schools.com/
W3Schools is a very popular website for learning the basics of web technology like HTML, CSS, and JavaScript. It's a great resourse because for someone wanting to get into web development or web design, this is the place to start learning the basics before going on to more advanced topics. And even for seasoned developers, it's good for use as a reference since the sheer number of HTML tags, CSS properties, and the nuances associated with each individual one make them hard to remember.

SitePoint http://reference.sitepoint.com/css
An less popular alternative to W3Schools with documentation on HTML, CSS, and JavaScript. It differs from W3Schools in that SitePoint points out feature support across browsers so the website doesn't break when you view it with another browser. It also mentions bugs and quirks specific to each browser for even more browser compatibility.

Mozilla's Developer Network (MDN) https://developer.mozilla.org/en-US/docs/Web
Another similar resource aimed more towards web developers rather than designers with its extensive coverage of JavaScript guides, tutorials, and documentation, although it has a good coverage of HTML and CSS as well. While the previous two websites cover the basics of web technologies, MDN covers the intermediate to advanced topics of JavaScript and CSS that those who has already grasped the basics can learn. After all, we can't talk usability without getting the website working first.

Stack Overflow http://stackoverflow.com/
Stack Overflow is a platform for questions and answers related to programming. Without a doubt every programmer has met a problem they don't know how to solve themselves, and this is the place to go for answers because chances are the same problem has already been encountered and solved before. In addition to programming, it has lots of questions and answers related to web design as well since the two are closely related.

Smashing Magazine http://www.smashingmagazine.com/
Unlike the previous resources that were oriented more towards programming, Smashing Magazine is for both web designers with articles on visual and user interaction aspects of web design. The guidelines listed on my website is far from comprehensive, and Smashing Magazine fills in some of the holes like designing for mobile and the intricacies of typography. It also extends on topics mentioned here, mainly the usability and the user experience sections.

Pixel Perfect Precision Handbook http://ustwo.comt/ppp/
The handbook covers digital design that extends beyond websites to apps and so on. The handbook tackles the usability issue from another angle, more from a design standpoint rather than user experience. The second half has much more detailed instructions on how to realize individual components of the UI, like layering buttons and text in Photoshop.

Learn Web Design Guide http://www.skilledup.com/learn-web-design-guide/
The guide is a curated collection of resources for learning web design, including tutorials, guides, online courses, references, and more. Only so much information can be contained in a single guide, or in a particular format. The collection enables extensive learning of web design with a wide variety of resources each covering different aspects of the topic, all nicely curated in one page.
